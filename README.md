# calculadoraM3

Proyecto Modulo 3 - Aplicación de consola.

A continuación se presenta una simulación de calculadora de IMC (Indice de Masa Corporal), que demostrará a través de la consola 
operaciones matemáticas básicas y funciones fundamentelas de el lenguaje JavaScript. Actualmente utilizamos la versión de ES6 para
este proyecto.

La manera mas óptima de ejecutar el código es descargando y extrayendo los archivos ZIP, 
abrirlos con el programa Visual Studio Code y ejecutarlo con la ayuda de la extensión "Live Server". 

Una vez abierto y ejecutado el código, el correcto flujo es 
abrir la consola DevTools (F12) y seguir las instrucciones: 

ATENCIÓN: Es muy importante, que al momento de abrir la consola se pueda 
escribir "iniciarCalculadora()", como lo indica el index. Ya que fue necesario agregarlo 
porque los prompt y las alertas intervenian con el correcto funcionamiento de la consola y/o 
el index y quise mantenerlas en el código. 

Una vez iniciado el proceso, debes completar con los datos solicitados para que 
la calculadora pueda realizar las operaciones matematicas requeridas 
en la consigna y arroje los resultados. 

Cabe mencionar, que en esta oportunidad y a diferencia de mi proyecto mi anterior, quise 
ayudarme con los comentarios (//) en el código JS para ordenar y explicar cada una de ellas. 
Esto me permitió crear mi calculadora con la lógica y cumpliendo los requisitos de el proyecto.
Por lo que, allí se encuentra en detalle la aplicación de los conceptos estudiados en el Módulo 3. 

Puedo concluir, que este proyecto me ayudó a comprender la sintaxis básica del lenguaje JS, como manejar 
las diferentes variables y el correcto uso de las estructuras condicionales. Como el uso de arrays es útil para 
múltiples elementos y que las funciones nos ayudan a organizar los datos de forma rapida y ordenada para su correcta ejecución.
